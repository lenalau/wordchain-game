Wortfällt

A word will fall down and you have to type it in with its correct spelling.
Before you type the next word, you have to repeat all the words, that passed before(correct order and spelling, separated by blanks).
With each new word, the speed decreases, to make sure that you have enough time, to type in all the words.
If you fail even one letter, you start again.



Example:
"Wort1"
"Wort1 Wort2"
"Wort1 Wort2 Wort3"
"Wort1 Wort2 Wort3 Wort4"
"Wort1 Wort2 Wort3 Wort4 Wort5"
"Wort1 Wort2 Wort3 Wort4 Wort5 Wort6"
"Wort1 Wort2 Wort3 Wort4 Wort5 Wort6 Wort7"
"Wort1 Wort2 Wort3 Wort4 Wort5 Wort6 Wort7 Wort8"
